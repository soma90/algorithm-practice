# algorithm-practice
프로그래밍 문제 풀이. 파일당 하나의 문제로 되어있고 문제 설명과 풀이 코드로 구성.

## 프로젝트 구성
├── basic : 코드스테이츠 기본레벨 문제
├── datastructure : 코드스테이츠 자료구조 관련 문제
├── immersive : 코드스테이츠 immersive레벨 문제
├── lv1 : 프로그래머스 lv1 문제
├── lv2 : 프로그래머스 lv2 문제
├── lv3 : 프로그래머스 lv3 문제
└── recursion : 코드스테이츠 재귀관련 문제 

## 사용한 기술 스택 및 라이브러리
* Javascript, Node.js
